# Инструкция по настройке GitHub Pages

## ⚠️ Важно: Используйте настройки репозитория (самый простой способ)

GitHub Actions может иметь ограничения из-за правил защиты окружения. Лучше использовать простой метод через настройки.

## Шаг 1: Активация через настройки репозитория

1. Перейдите в настройки репозитория:
   **https://github.com/richboy13/Islam/settings/pages**

2. В разделе "Build and deployment":
   - **Source**: выберите `Deploy from a branch`
   - **Branch**: выберите `main`
   - **Folder**: выберите `/ (root)` 
   - Нажмите **Save**

3. Подождите 1-2 минуты

## Шаг 2: Проверка

После сохранения настроек сайт будет доступен по адресу:
**https://richboy13.github.io/Islam/**

## Альтернатива: Если нужно использовать docs папку

Если хотите использовать папку `docs`:

1. В настройках Pages выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/docs`
   - Нажмите **Save**

## Проверка статуса

- После настройки подождите 1-2 минуты
- Проверьте, что репозиторий публичный (не private)
- Очистите кэш браузера если сайт не появляется

## Структура файлов

- `index.html` - основной файл слайдов (в корне)
- `docs/index.html` - копия файла (в папке docs)
- `.nojekyll` - файл для отключения Jekyll обработки

## Если сайт не работает

1. Убедитесь, что репозиторий публичный
2. Проверьте, что файл `index.html` существует в выбранной папке
3. Подождите несколько минут после настройки
4. Очистите кэш браузера (Ctrl+Shift+R или Cmd+Shift+R)
